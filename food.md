# 乡愁

> 味觉是乡愁最顽固的记忆——离家越远，那碗糁饭越香。

靖远/平川地处黄河上游，是丝绸之路上的一方沃土。靖远素有**"塞上江南"**"陇上名邑""黄河明珠"之美誉，黄河穿城 154 公里，是甘肃段流经里程最长的县。丰饶的水土孕育了独具特色的饮食文化。

## 硬菜三绝

靖远是**"羊羔肉美食之乡"**，也是甘肃省最大的羊羔肉产地和集散地。本地饮食以面食为主、融合杂粮，口味偏重，讲究实惠，其中三道"硬菜"名震陇原：

### 🐑 靖远羊羔肉

"名震塞北，誉满陇原"。靖远羊羔肉是国内首例以哺乳类动物注册的**中国国家地理标志商标**，肌肉纤维细、嫩度高，膻味轻微、香味浓郁。可爆炒、红烧、黄焖、干炸、烧烤、清炖、清蒸，辅以 10 多种中药材为佐料，每种做法都是一绝。

### 🐴 东湾驴肉

东湾镇特色招牌菜，色鲜味美，制作讲究**"一泡二焯三慢煮"**，风味独特，是外地人到靖远必尝的硬菜。

### 🥘 酸烂肉

又称"酸菜粉条炒肉"，起源于农家的家常小菜。用腌制好的酸白菜切丝，与土豆粉条、大肉一起爆炒，酸香扑鼻、实惠味美。配上一碗黄米糁饭，就是最地道的家乡味道。如今已从农家灶台登上大雅之堂，成为招待宾客的佳肴——也是每个靖远/平川人离家后最想念的一口。

## 主食与家常风味

| 类别 | 代表 |
|------|------|
| **蒸食** | 蒸馍、花卷、马蹄子、月饼、包子、发糕、黄米甜馍馍 |
| **煮食** | 长面（臊子面）、截截子、旗花子、面片子、饺子、**糁饭**、荞面搅团 |
| **烙食** | 锅盔（外酥内软，便于储存）、大饼、**一窝丝**（细如发丝，口感酥香） |
| **炸食** | 油饼、油条、麻花、酥合子、糖油糕 |
| **酿皮** | 西北经典凉食，爽滑筋道，夏日解暑佳品 |
| **饮品** | 罐罐茶、**浆水**（芹菜发酵，清暑解毒）、醪糟、甜酒醅、麦茶、枣儿茶 |

### 时令野菜

春夏之际，苦苣菜、野韭、野葱、苜蓿、地软软、枸杞芽、榆钱、槐花等轮番登场，是大自然对这片土地的馈赠。

### 宴席与节庆

- **婚宴**：讲究"**十三花**"宴席，十三道炒菜依次上桌，寓意吉祥圆满
- **节庆**：花馍馍以姜黄、香豆子、红曲巧作，造型如菊、荷、佛手，又称"点心"，是走亲访友的体面礼物

## 七大地标

靖远拥有 **7 个国家地理标志保护产品**，获评**全国首批农产品质量安全县**：

> 🏅 小口大枣 · 靖远枸杞 · 靖远黑瓜籽 · 靖远羊羔肉 · 靖远文冠果油 · 靖远旱砂西瓜 · 大庙香水梨

同时享有"中国文冠果之乡""陇原蔬菜之乡""枸杞之乡""籽瓜之乡"等称号。
